AI Project

1. Prerequisites


.winget         - install winget in Windows
.git            - install git in Windows, use winget to install
.vscode         - install vscode in Windows, download or use winget
.wsl            - install wsl in Windows
.powershell     - install powershell in Windows, use winget to install
.github account - create an account at github.com, could use google account
