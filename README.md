## 1. install vscode

 https://code.visualstudio.com/Download#
* double click to install 

## 2. install Python
* download from 
 https://www.python.org/ftp/python/3.10.1/python-3.10.1-amd64.exe
+ tick "Add to Path"

* in vscode
-> Extension, search python and install

## 4. Python
* run in shell 
+ check version 
> python --version

+ start python shell 
> python 
>>> print("Hello world!")
>>> print( 4 + 5 )
>>> exit() 

* python script
+ create a file, hello.py
```python

print(4 + 5)
```
+ run the script 
> python .\hello.py
